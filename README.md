This repository includes the Excelsheet and Rmarkdown to process the metadata from different CLIP methods to generate Figure4. In order to genearte the Figure4,clone the repository and run the R-markdown that will generate and save the Beeswarm plot for Figure 4 in the working directory. The source of different CLIP methods are provided as follows:

1) TLC-CLIP: https://academic.oup.com/nar/article/51/13/e70/7191424#supplementary-data

2) eCLIP,PARCLIP,PAR-iCLIP,CLIP: https://www.nature.com/articles/nmeth.3810#Sec21
