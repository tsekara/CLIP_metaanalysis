This repository includes the Excelsheet and Rmarkdown to process the metadata from different CLIP methods. The source for the metadata are provided as follows:

1) iCLIP: https://static-content.springer.com/esm/art%3A10.1038%2Fnsmb.1838/MediaObjects/41594_2010_BFnsmb1838_MOESM21_ESM.pdf

2) irCLIP: https://static-content.springer.com/esm/art%3A10.1038%2Fnmeth.3840/MediaObjects/41592_2016_BFnmeth3840_MOESM217_ESM.pdf

3) TLC-CLIP: https://academic.oup.com/nar/article/51/13/e70/7191424#supplementary-data

4) eCLIP,PARCLIP,PAR-iCLIP,CLIP: https://www.nature.com/articles/nmeth.3810#Sec21

The "Total Input reads","Genome aligned reads" and "Usable reads" from the above sources were compiled into "CLIP_data_compiled.xlsx" file that were processed using "CLIP_stats.Rmd" to generate the Figure 4.
